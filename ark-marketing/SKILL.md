## 模組 1：ASO（應用商店優化）

### 核心要素

| 要素 | 優化方向 |
|------|---------|
| 標題 | 品牌名 + 核心關鍵字（30 字內） |
| 副標題 | 差異化賣點（iOS 30 字） |
| 關鍵字 | 100 字元（iOS），長尾優先 |
| 截圖 | 前 3 張決定下載，展示核心玩法 |
| 影片 | 15-30 秒，前 5 秒抓注意力 |
| 描述 | 前 3 行最重要（折疊前可見） |

### SOP

1. 競品關鍵字分析（前 10 名同類遊戲）
2. 長尾關鍵字挖掘（搜尋量 vs 競爭度）
3. A/B 測試截圖（每 2 週迭代）
4. 追蹤排名變化（每日）

## 模組 2：CRO（轉換率優化）

### 漏斗分析

```
曝光 → 商店頁瀏覽 → 下載 → 首次開啟 → 教學完成 → Day1 留存 → 付費
```

### 各階段優化

| 階段 | 指標 | 優化手段 |
|------|------|---------|
| 曝光→瀏覽 | CTR | 圖示 + 標題吸引力 |
| 瀏覽→下載 | CVR | 截圖 + 影片 + 評分 |
| 下載→開啟 | 啟動率 | 推播提醒 + 包體大小 |
| 開啟→教學 | 完成率 | 教學流程簡化 |
| 教學→留存 | D1 | 首日體驗 + 獎勵 |
| 留存→付費 | 付費率 | 時機 + 定價 + 價值感 |

## 模組 3：文案撰寫

### 遊戲文案類型

| 類型 | 長度 | 重點 |
|------|------|------|
| 商店描述 | 4000 字 | 前 3 行 hook + 功能列表 |
| 推播通知 | 50 字 | 緊迫感 + 個人化 |
| 活動公告 | 200 字 | 獎勵明確 + 時間限制 |
| 社群貼文 | 100 字 | 互動問句 + 視覺素材 |
| 廣告文案 | 30 字 | 痛點 + 解法 + CTA |

### 文案公式

- **AIDA**：注意 → 興趣 → 慾望 → 行動
- **PAS**：問題 → 激化 → 解決方案
- **BAB**：之前 → 之後 → 橋樑

## 模組 4：UA（用戶獲取）

### 管道選擇

| 管道 | CPI 範圍 | 適合 |
|------|---------|------|
| Facebook/Meta | $1-5 | 廣泛受眾 |
| Google UAC | $0.5-3 | 自動化投放 |
| TikTok | $1-4 | 年輕受眾 |
| Unity Ads | $0.3-2 | 遊戲玩家 |
| KOL/網紅 | 變動 | 品牌建立 |

### 素材策略

- 前 3 秒決定觀看（hook）
- 展示核心玩法（非 CG）
- 加入 UGC 風格素材
- 每週更新 3-5 組素材（抗疲勞）

## 模組 5：LiveOps 活動規劃

### 活動節奏

| 頻率 | 類型 | 目的 |
|------|------|------|
| 每日 | 登入獎勵 / 每日任務 | 習慣養成 |
| 每週 | 週末活動 / 排行榜 | 活躍提升 |
| 每月 | 主題活動 / 新內容 | 回流 + 付費 |
| 季度 | 大版本 / 聯動 | 話題 + 新用戶 |

### 活動設計 SOP

1. 確定目標（留存/付費/活躍）
2. 設計獎勵結構（免費層 + 付費層）
3. 設定時間窗口（FOMO）
4. 準備素材（公告/推播/社群）
5. 上線監控（參與率/完成率/營收）
6. 事後分析（vs 預期，歸檔經驗）

## 模組 6：留存策略

### 關鍵指標

| 指標 | 業界基準（Slot） | 目標 |
|------|----------------|------|
| D1 | 35-45% | ≥ 40% |
| D7 | 15-20% | ≥ 18% |
| D30 | 5-8% | ≥ 7% |

### 留存手段

- 新手引導優化（3 分鐘內體驗核心玩法）
- 每日登入獎勵（遞增式）
- 推播策略（個人化 + 時機）
- 社交功能（好友/公會/排行）
- 內容更新節奏（每 2 週新內容）
