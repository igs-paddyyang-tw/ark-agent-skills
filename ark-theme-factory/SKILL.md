# 主題工廠

提供精選的專業字型與色彩主題集合，每個主題都有精心挑選的色盤和字型搭配。
選定主題後，可套用到任何 artifact。

## 用途

為投影片、文件或其他 artifact 套用一致、專業的樣式。每個主題包含：
- 連貫的色盤（含 hex 色碼）
- 互補的標題/內文字型搭配
- 適合不同情境和受眾的獨特視覺識別

## 使用流程

1. **展示主題目錄**：顯示 `theme-showcase.pdf` 讓使用者瀏覽所有可用主題
2. **詢問選擇**：詢問要套用哪個主題
3. **等待確認**：取得明確的主題選擇確認
4. **套用主題**：將選定主題的色彩和字型套用到 artifact

## 主題詳情

每個主題定義在 `themes/` 目錄，包含完整規格：色盤（hex 色碼）、字型搭配、視覺識別。

## 套用流程

選定主題後：
1. 從 `themes/` 目錄讀取對應主題檔案
2. 將指定的色彩和字型一致套用到整個 artifact
3. 確保適當的對比度和可讀性
4. 在所有頁面維持主題的視覺識別

## 自訂主題

當現有主題都不適合時，可建立自訂主題：
- 根據提供的輸入產生新主題
- 給主題一個描述性名稱
- 使用基本描述選擇適當的色彩/字型
- 產生後展示供審查確認
- 確認後按上述流程套用
