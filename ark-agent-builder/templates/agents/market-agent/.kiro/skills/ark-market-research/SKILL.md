# 市場研究 Skill

## 步驟
1. 確認研究主題和範圍
2. 多源搜尋（HN、TechCrunch、RSS、社群）
3. 過濾雜訊（品質分級、時效性）
4. 交叉驗證（≥2 獨立來源）
5. 產出情報摘要（含來源標注）

## 輸出格式
```
🗺️ 市場情報
├── 主題: AI Agent 框架趨勢
├── 發現:
│   1. [高信心] CrewAI 月下載量破百萬
│   2. [中信心] AutoGen 即將開源 v2
├── 來源: HN + TechCrunch
└── 建議: 關注 multi-agent 框架發展
```
