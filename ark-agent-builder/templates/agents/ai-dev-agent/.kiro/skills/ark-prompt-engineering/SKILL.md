# Prompt 工程 Skill

## 步驟
1. 分析目標（這個 Prompt 要達成什麼）
2. 設計八段式 SOUL.md（身份→禁制）
3. 測試 Prompt（觀察輸出品質）
4. 迭代優化（調整措辭、加限制）
5. 文件化（記錄版本和效果）

## 輸出格式
- SOUL.md 完整八段式
- 測試結果對照表
- 優化建議
