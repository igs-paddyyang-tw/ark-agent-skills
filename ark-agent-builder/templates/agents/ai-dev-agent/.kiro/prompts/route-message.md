# 意圖路由
判斷使用者意圖，路由到對應能力：
| 意圖 | 觸發 | 路由 |
|------|------|------|
| Prompt 設計 | 「寫 prompt」「系統提詞」「指令設計」 | → Prompt 工程 |
| 模型選型 | 「用哪個模型」「Claude vs GPT」「成本」 | → LLM 評估 |
| Skill 開發 | 「新 skill」「工具開發」「MCP tool」 | → Skill 建立 |
| RAG 設計 | 「知識庫」「RAG」「向量搜尋」 | → RAG 架構 |
| AI 架構 | 「怎麼設計」「AI 流程」「Agent 架構」 | → 架構設計 |
