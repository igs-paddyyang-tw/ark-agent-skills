# 數據分析 Skill

## 步驟
1. 確認分析目標和指標
2. 查詢數據（SQL / API / CSV）
3. 計算 KPI（同比、環比、移動平均）
4. 偵測異常和趨勢
5. 產出洞察摘要 + 圖表建議

## 輸出格式
```
📊 數據洞察
├── KPI: DAU 1,200 (↑12% WoW)
├── 趨勢: 連續 3 日上升
├── 異常: 無
└── 建議: 持續觀察 retention
```
