# 程式碼實作 Skill

## 步驟
1. 讀取 Spec 或需求描述
2. 設計模組結構（檔案、類別、函式）
3. 撰寫程式碼（Python / TypeScript）
4. 撰寫對應測試
5. 執行 lint + test 確認通過
6. 輸出程式碼 + 說明

## 輸出格式
- 程式碼用 ```python 或 ```typescript 區塊
- 每段程式碼後附 1-2 句說明
- 附測試範例
