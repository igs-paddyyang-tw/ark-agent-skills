本 Skill 指導產出獨特、生產級品質的前端介面，避免千篇一律的「AI 罐頭風格」。
實作真正可運行的程式碼，對美學細節與創意選擇給予極高關注。

使用者提供前端需求：元件、頁面、應用程式或介面。可能包含用途、受眾或技術限制等背景資訊。

## 設計思維

在寫程式碼之前，先理解情境並確立大膽的美學方向：
- **目的**：這個介面解決什麼問題？誰在使用？
- **調性**：選擇一個極端風格：極簡主義、極繁主義、復古未來、有機自然、奢華精緻、趣味玩具風、雜誌編輯風、粗獷主義、裝飾藝術幾何風、柔和粉彩、工業實用風等。以此為靈感，設計出忠於美學方向的作品。
- **限制**：技術需求（框架、效能、無障礙）。
- **差異化**：什麼讓這個設計令人難忘？使用者會記住的那一件事是什麼？

**關鍵**：選擇清晰的概念方向並精準執行。大膽的極繁主義和精緻的極簡主義都可行——重點是意圖性，而非強度。

然後實作可運行的程式碼（HTML/CSS/JS、React、Vue 等），必須：
- 生產級品質且功能完整
- 視覺上引人注目且令人難忘
- 具有清晰美學觀點的一致性
- 每個細節都經過精心打磨

## 前端美學指南

重點關注：
- **字型**：選擇美觀、獨特、有趣的字型。避免 Arial、Inter 等通用字型；選擇能提升美感的獨特字型。搭配一個有特色的標題字型與精緻的內文字型。
- **色彩與主題**：確立一致的美學風格。使用 CSS 變數確保一致性。主色搭配鮮明強調色，優於膽怯的均勻分佈色盤。
- **動態效果**：使用動畫做效果與微互動。HTML 優先使用純 CSS 方案。React 可用 Motion 函式庫。聚焦高影響力時刻：一個精心編排的頁面載入動畫（搭配 animation-delay 的交錯顯示）比零散的微互動更令人愉悅。善用捲動觸發和令人驚喜的 hover 狀態。
- **空間構圖**：出人意料的排版。不對稱。重疊。對角線流動。打破網格的元素。大量留白或有控制的密度。
- **背景與視覺細節**：營造氛圍與深度，而非預設純色背景。加入符合整體美學的情境效果與紋理。運用漸層網格、雜訊紋理、幾何圖案、多層透明、戲劇性陰影、裝飾邊框、自訂游標、顆粒覆蓋等創意形式。

絕對不要使用千篇一律的 AI 生成美學：過度使用的字型家族（Inter、Roboto、Arial、系統字型）、老套的配色方案（特別是白底紫色漸層）、可預測的排版與元件模式、缺乏情境特色的罐頭設計。

創意詮釋，做出令人意外的選擇，讓設計真正為情境量身打造。每個設計都應該不同。在淺色與深色主題、不同字型、不同美學之間變化。絕對不要在多次產出中收斂到相同的選擇。

**重要**：實作複雜度要匹配美學願景。極繁主義設計需要精心的程式碼、大量動畫與效果。極簡或精緻設計需要克制、精準，以及對間距、字型和微妙細節的仔細關注。優雅來自於對願景的良好執行。
